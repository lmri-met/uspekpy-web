USpekPy Web

USpekPy web is a web application that allows you to compute mean radiation protection quantities for a simulated x-ray spectrum with uncertainties using Monte Carlo techniques. It allows to compute the mean values and uncertainties of the following integral quantities: the first and second half-value layer for aluminium and copper, the mean energy, the air kerma and the mean air kerma-to-dose-equivalent conversion coefficient. USpekPy web is based on USpekPy and SpekPy Python packages. USpekPy web allows you to simulate a single case or a batch of cases. A single case is defined for a specific x-ray radiation quality and a specific operational magnitude at a specific irradiation angle.
